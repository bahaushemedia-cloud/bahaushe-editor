# BAHAUSHE Editor — Android App

A native Android app for writing and transliterating Hausa text in BAHAUSHE script.

## Features
- Custom BAHAUSHE script keyboard
- Hausa Latin to BAHAUSHE auto-transliteration
- Dual-pane mode (Latin + Script side by side)
- Local document storage
- JSON export and import
- 3-way language toggle: English / Hausa Latin / Hausa Script
- Print and PDF export

## Build
This app is built automatically via GitHub Actions.
Go to the Actions tab to download the APK.
